Local RNA-seq analysis, no hosting bill

AutoRNAseq

A browser-based RNA-seq workflow that runs on the user's own computer with Docker. Open the app locally, keep data private, and move from FASTQ files to differential expression, enrichment, PPI, and biomarker discovery without coding.

Download latest release Quick start

After installation, the app opens at http://localhost:3838.

Quick start for non-coders

  1. Download the latest release from GitHub.
  2. Install Docker Desktop if it is not already installed.
  3. Run the included startup script for Windows or macOS/Linux.
  4. Open http://localhost:3838 in your browser.
  5. Upload your FASTQ files and metadata, then follow the sidebar modules.

What it does

Quality control, trimming, alignment, quantification, DESeq2 analysis, GO/KEGG/GSEA, STRINGdb PPI networks, and machine-learning biomarker ranking.

Why local

No central server costs, no uploading large raw sequencing files to a hosted service, and a consistent containerized environment for reproducibility.

Best for

Researchers, students, and labs that need a guided transcriptomics workflow without maintaining their own web infrastructure.
